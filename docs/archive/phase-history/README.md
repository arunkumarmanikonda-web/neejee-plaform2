# Archived Phase Planning Documents

This folder contains superseded planning documents preserved for historical reference.

## Status
These files are **not** the active execution source of truth.

## Current source of truth
Use the reconciled phase tracker / updated tracker workbook as the current program-status reference.

## Why these files were archived
Earlier planning documents were useful during earlier implementation windows, but they no longer reflect the reconciled status after:
- Phase 0 carryover review
- later DOCX work-log reconciliation
- completion of Phase 1 work
- completion of P2-004 catalogue export pipeline
- completion of editorial-block AI automation
- removal of stale pending items already closed in later execution

## Typical archived contents
- old phase scope docs
- intermediate roadmap versions
- half-phase planning notes such as `PHASE1_5_*`
- superseded operational planning drafts

## Archive policy
When a planning document is no longer authoritative but still useful for history:
1. move it into this archive area
2. do not delete historical context unless it is accidental junk
3. prefer preserving filename history
4. avoid mixing archived plans with active planning docs

## Notes
If a legacy doc must remain in its original location temporarily, add a banner at the top:

> Archived / superseded document
>
> This file is preserved for historical reference only.
> The reconciled tracker is the active source of truth.

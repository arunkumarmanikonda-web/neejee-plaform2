# Neejee Delivery Roadmap

This repository uses a single active roadmap source of truth.

## Source of truth
The live delivery tracker is the blueprint workbook:
- https://www.genspark.ai/api/files/s/rqJXTNKi

## Roadmap policy
- Status changes are recorded in the tracker.
- Archived PHASE1 / PHASE1_5 documents are historical only.
- No older phase-numbering document should be treated as active planning.
- Go-live requires blueprint completion, not partial milestone completion.

## Execution order
1. Phase 0 — Closeout and stabilization
2. Phase 1 — Build the Read Layer
3. Phase 2 — Catalogue V1
4. Phase 3 — ERP shadow integration
5. Phase 4 — Catalogue V2 and seller onboarding
6. Phase 5 — ERP operational maturity
7. Phase 6 — Advanced intelligence

## Current rule
If a roadmap statement conflicts with the tracker workbook, the tracker workbook wins.

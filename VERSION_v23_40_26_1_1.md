# v23.40.26.1.1 — AI-Driven Taxonomy Wiring (Sprint 26.1.1)

## What changed

### NEW: AI Category Resolver (`lib/taxonomy/resolver.ts`)
Single source of truth that turns any string ("Banarasi", "Floor lamp", "Attar")
or product context into a leaf category. If the leaf or sub-category doesn't
exist yet, the AI invents a sensible name and auto-creates it (aiGenerated=true).
Used by the picker, bulk import, and migration job — same code path everywhere.

### NEW: Searchable Autocomplete CategoryPicker (`components/admin/CategoryPicker.tsx`)
- Type any term → see full breadcrumb inline, e.g.
    "Women / Sarees / Banarasi"
    "Women / Dupattas & Stoles / Banarasi"  (separate option)
- "✨ Resolve with AI" button: passes product name/description/craft/material to
  the AI which picks the right leaf — and auto-creates it if missing.

### NEW: Endpoints
- `GET  /api/admin/taxonomy/search?q=…`        — picker autocomplete
- `POST /api/admin/taxonomy/ai-resolve`         — resolve free text → leaf (auto-creates)
- `POST /api/admin/taxonomy/migrate-products`   — batch re-categorise existing products

### NEW: Admin UI — `/admin/taxonomy/migrate`
- Dry-run preview → see exactly what would change before committing.
- Per-row breakdown showing OLD → NEW path, match method (exact / fuzzy / ai /
  ai-created), and any categories created on the fly.
- Optional "Regenerate SKU to match new taxonomy" toggle
  (`NEE-MAIN-SUB-LEAF-####`).
- Linked from /admin/taxonomy top-right ("✨ AI MIGRATE PRODUCTS").

### UPDATED: Product create / edit forms now use the picker
- Admin: `/admin/products/new`, `/admin/products/[id]`
- Seller: `/seller/products/new`

### UPDATED: Bulk Excel import (`/api/admin/products/import`)
- Category column accepts slug, full path (`women/sarees/banarasi`), or free
  text. If exact match fails, the AI resolver fills the gap and (with the
  admin's import permission) auto-creates the leaf.
- SKU now generated by `generateSkuFor(categoryId)` →
  `NEE-WOM-SAR-BAN-####` style.
- Per-row result message reports any AI-created categories.

### UPDATED: Excel template + export
- Template `Category` column note now explains AI fallback.
- Export adds a new column **Category breadcrumb** (`women/sarees/banarasi`)
  alongside the existing slug.
